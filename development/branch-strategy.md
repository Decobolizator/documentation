# Gestion des branches Git

## Branches principales

- `main` : branche de production, contient le code stable et prêt à être déployé.
- `dev` : branche de développement, contient les dernières fonctionnalités en cours de développement.
- `feat/*` : branches de fonctionnalités, créées à partir de `dev` pour développer une nouvelle fonctionnalité spécifique.
- `fix/*` : branches de correction de bugs, créées à partir de `dev` pour corriger un bug spécifique.
